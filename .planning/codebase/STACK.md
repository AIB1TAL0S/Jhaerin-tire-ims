# Tech Stack

## Core
- **Framework:** SvelteKit v2 with Svelte v5 (Runes enabled)
- **Language:** TypeScript
- **Build Tool:** Vite v8
- **Runtime Environment:** Node.js (v22.x targeted via Vercel Adapter)

## Frontend
- **Styling:** TailwindCSS v4
- **Component Library:** shadcn-svelte
- **Forms & Validation:** sveltekit-superforms paired with Zod
- **Icons:** @lucide/svelte
- **Charts:** layerchart

## Backend / Database
- **ORM:** Drizzle ORM
- **Database:** PostgreSQL (via postgres.js)
- **Authentication:** Supabase Auth (SSR via @supabase/ssr)

## Tooling & Quality
- **Linter / Formatter:** ESLint, Prettier (with Svelte and Tailwind plugins)
- **Testing:** Vitest, Playwright (for browser testing)
- **Deployment:** Vercel Adapter
